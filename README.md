# Prodigy

Prodigy is the annual symposium of Production Department at NIT Trichy. 

For Prodigy 2023, we built a website using:

- NextJS
- Tailwindcss
- Shadcn/ui
- Magicui

## About

To setup the project locally:

```bash
git clone <url>
cd Prodigy-Webops
npm install
npm run dev
```
